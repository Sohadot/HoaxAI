# Public Executive Overview Standard v1

Sprint 104 — **Decision:** DEC-122

## Executive Overview Standard Statement

Executive-overview routes provide concise public high-level orientation. Minimum 750 visible words per route. Required sections: Reference summary, Overview purpose, Executive reading path, What this overview supports/does not claim, Reference Answer, Source Confidence, Cite This Reference, Retrieval Capsule, boundaries, page-end navigation.

## Requirements

- Anchors: reference-summary, overview-purpose, executive-reading-path, reference-answer, source-confidence, cite-this-reference, retrieval-capsule, boundary
- Link to `/executive-overview/`, 3+ strategic/external/reviewer routes, 5+ utility/reference/pathway routes, 2+ sibling overview routes
- No private data-room access, downloadable reports, pricing, transactions, detector claims, JS, or forms
- Stale route-count language (58/63/68) prohibited in public HTML when describing current surface
