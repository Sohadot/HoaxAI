# Public Reference Citation Orientation v1

**Decision:** DEC-137  
**Sprint:** Sprint 120  
**Status:** Adopted — visible production route

## Route governed by this decision

This sprint adds one visible production route:

- `/citation-orientation/` (ROUTE-0103, PUB-FILE-0103)

DEC-137 governs this route directly in Sprint 120.

## Governing sentence

Citation orientation helps humans and AI agents cite Hoax.ai reference pages safely; it does not create authority claims, verification claims, generated citations, legal citations, academic endorsement, detector support, score support, or verdict support.

## Purpose

Public Reference Citation Orientation v1 teaches citation-safe reference use for Reference Answer, Source Confidence, Cite This Reference, Retrieval Capsule, and boundary blocks across the 103-route public surface.

## Citation orientation themes

1. Citation-safe use of Hoax.ai
2. Citing Reference Answer blocks
3. Citing Source Confidence blocks
4. Citing Cite This Reference blocks
5. Citing Retrieval Capsule blocks
6. Citation without verdict language
7. Citation without detector language
8. Citation support versus case conclusion
9. Citation and AI agents
10. Citation and strategic review

## Boundary

This route is citation orientation only. It does not introduce citation-generator behavior, generated-citation output, legal citation claims, academic endorsement, authority claims, verification claims, detector support, score support, verdict support, workflow behavior, API behavior, upload behavior, form behavior, report generation behavior, dashboard behavior, or transaction behavior.

## Production-first compliance

This governance file is tied to the visible Sprint 120 production route `/citation-orientation/`. It is not abstract preparatory governance.
