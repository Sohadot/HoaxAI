Condition definition

An evidence condition is a named interpretive state describing what can and cannot be responsibly read from available material at review time. Evidence conditions are not conclusions, not authenticity labels, and not enforcement recommendations.

Hoax.ai publishes evidence conditions as a public reference library so humans and AI agents can classify reading limits before belief. The library sits alongside core concepts, deep concepts, pathways, audience paths, route-group deepening, review layers, and system-map routes.

Each child page defines one condition, explains human and retrieval use, relates sibling conditions, and preserves explicit boundary negations. No condition page authorizes upload capability, scoring surfaces, dashboards, graph tools, or automated report generation.
