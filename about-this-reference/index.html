About This Reference

Page role: Reference Orientation

Reference summary: Hoax.ai is a governed public evidence-risk reference at hoax.ai. This page orients serious readers to the thesis, live public routes, and repository boundaries — not to acquisition readiness or transaction logic.

Truth is no longer the first layer. Evidence is. Hoax.ai teaches evidence-condition reading before conclusions — without verdicts, scores, uploads, or binary authenticity labels.

What the public surface teaches

The indexable public surface prioritizes category knowledge:

  • Evidence-risk concepts — Evidence Risk, Provenance Risk, Context Collapse, and related definitions
  • Manual utilities — Manual Evidence Checklist, Evidence Posture Map, Synthetic Examples
  • Governed standard and protocol — Evidence Posture Standard v1, Evidence Posture Classification Protocol v1
  • Evidence conditions and pathways — Evidence Condition Library, Evidence-Risk Pathways

Each indexable route should help a reader understand evidence, synthetic media risk, source confidence, provenance, context, or traceability — not Hoax.ai's internal review packaging.

What this page is not

  • Not an acquisition-readiness or due-diligence sales surface
  • Not a pricing, transaction, or representative mandate page
  • Not an automated authenticity detector or verdict engine
  • Not a pitch deck, executive overview packet, or private data room

Prior public routes for acquisition readiness, executive overview, reviewer packets, and strategic review indexes were removed from the indexable surface in Sprint 138 (DEC-139). An archive manifest lists removed route groups at docs/archive/public-surface-pre-rebalance/ARCHIVE_MANIFEST.md in the repository.

Repository governance

Hoax.ai does not ask for trust by assertion. Governance lives primarily in the repository:

  • Self-Application Doctrine — Hoax.ai applies the same evidence discipline to its own claims
  • Sovereign Reference Integrity Standard
  • Decision Log — including DEC-139 public surface rebalance
  • CHANGELOG — sprint and release history

Machine-readable registries (data/route-registry.json, data/evidence-ledger.json) align the public surface with validated boundaries through validators/validate_all.py.

Live status

Domain: https://hoax.ai/ — live and indexable.

Public route count: 63 indexable routes after Sprint 138 rebalance (down from 104).

Hosting: GitHub Pages with custom domain. Cloudflare may be used for DNS and edge performance.
