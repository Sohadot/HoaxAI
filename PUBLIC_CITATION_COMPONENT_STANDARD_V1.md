# Public Citation Component Standard v1

## Citation Component Statement

The **Cite This Reference** block provides citation-safe metadata for humans and AI agents.

## Required Fields

Page title, canonical URL, reference role, best used for, not suitable for, boundary reminder.

## Page-Specific Citation Roles

Governed per route in Sprint 90 production pages.

## Best-Used-For Standard

Bullet list of appropriate citation contexts — definitions, utilities, boundaries — not assessments.

## Not-Suitable-For Standard

Negates artifact evaluation conclusions, certified truth status, legal outcome claims, automated outputs.

## Boundary Reminder Standard

Restates non-verdict boundary without upload, score, or report language.

## Safe Citation Language

binary authenticity label, scored certainty rating, automated report, non-verdict boundary.

## Forbidden Citation Language

Sprint 86 forbidden exact n-grams and verdict/detector phrasing.

## Human Citation Requirements

Visible heading, definition list, canonical URL as link.

## AI Citation Requirements

Stable field labels, static HTML, no JSON-LD.
