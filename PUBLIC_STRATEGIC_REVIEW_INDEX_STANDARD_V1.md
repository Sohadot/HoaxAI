# Public Strategic Review Index Standard v1

Sprint 106 — Public Reference Strategic Review Index v1  
**Decision:** DEC-124  
**Date:** 2026-06-26

## Strategic Review Index Standard Statement

Strategic review index pages organize public inspection by strategic question without creating conclusions, scores, transaction claims, private access, reports, or operational claims.

## Required page structure

Reference summary, Index purpose, Strategic review path, What this index supports, What this index does not claim, Reference Answer, Source Confidence, Cite This Reference, Retrieval Capsule, Related Routes, page-end navigation, Boundary reminder, Non-transactional review boundary.

## Required index components

Stable anchors: reference-summary, index-purpose, strategic-review-path, reference-answer, source-confidence, cite-this-reference, retrieval-capsule, boundary.

## Strategic review path requirements

Five to seven linked steps through existing Hoax.ai routes per page.

## Route-group indexing requirements

Hub links all child routes; child routes link hub and siblings; homepage links all five routes.

## Human readability requirements

At least 850 visible words per new route; citation-safe wording; one H1 per page.

## AI readability requirements

Reference Answer, Source Confidence, Cite This Reference, Retrieval Capsule on every page.

## Citation/retrieval requirements

Canonical URLs, meta descriptions, Open Graph title and description on every page.

## Internal link requirements

Link to `/strategic-review/`, at least three strategic layer routes, at least five reference/utility/pathway routes, at least two sibling strategic-review routes.

## Non-verdict requirements

No detector claims, verdict language, upload, or automated report promises.

## Non-transactional requirements

No pricing, transaction, legal, or financial representation.

## Pitch-deck prohibition

Safe negative language required; no pitch-deck framing.

## Sales-page prohibition

Safe negative language required; no sales-page framing.

## Private data-room prohibition

No private data-room or due-diligence room claims.

## Downloadable report prohibition

No downloadable report behavior or claims.

## Forbidden regression patterns

Forbidden positive phrases in public HTML unless clearly negated.

## Thin-page prevention rules

Minimum 850 visible words per strategic-review index route.
