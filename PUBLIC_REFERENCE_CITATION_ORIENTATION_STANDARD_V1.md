# Public Reference Citation Orientation Standard v1

**Decision:** DEC-137  
**Route:** `/citation-orientation/`

## Standard scope

This standard governs the visible Sprint 120 citation-orientation route only.

## Required production properties

- Static HTML with one H1
- Minimum 2,000 visible words
- Canonical URL, meta description, and Open Graph tags
- Twenty-five required sections with stable anchors
- Ten citation-orientation themes
- Reference Answer, Source Confidence, Cite This Reference, Retrieval Capsule, boundary reminder, and non-transactional review boundary blocks
- No JavaScript, forms, uploads, or API behavior

## Prohibited behaviors

- Citation-generator behavior and generated-citation output
- Legal citation claims and academic endorsement
- Authority claims and verification claims
- Detector support, score support, and verdict support
- Workflow, operational procedure, and report-generation behavior
- Dashboard, graph-tool, score-card, and rating-system behavior
- Transaction, sales-page, consulting-funnel, due-diligence-room, and legal/financial representation behavior

## Navigation requirements

Homepage, system map, retrieval index, and reading sequences must link to `/citation-orientation/` where citation-safe orientation is relevant.
