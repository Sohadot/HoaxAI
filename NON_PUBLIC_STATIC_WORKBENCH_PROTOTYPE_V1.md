# Non-Public Static Workbench Prototype v1

**Version:** v1.0.0  
**Status:** non_public_static_workbench_prototype_v1_created  
**Maturity:** static_internal_prototype_only_no_engine_no_classifier_no_tool_no_public_route  
**Decision:** DEC-052

## A. Purpose

This document records the creation of the first non-public static Evidence Posture Workbench prototype.

## B. Non-Purpose

This sprint does **not** create a public workbench, public route, sitemap entry, public navigation, deployable product, engine, classifier, detector, upload functionality, scoring, fake/real outputs, forms, user inputs, analytics, API, monetization, DNS/Cloudflare changes, custom domain launch, or deployment authorization.

## C. Governing Principle

**A static prototype may show future form. It must not create future function.**

**The prototype must make Hoax.ai's evidence chamber visible without making Hoax.ai look like a detector.**

## D. Prototype Definition

The prototype is a static, non-operational, repository-visible but non-public web-surface artifact that visually explores the future Evidence Posture Workbench layout, evidence chamber identity, governed evidence field background, artifact-first structure, boundary rails, provenance shadows, refusal gate, not-assessable protected state, output envelope containment, and verification path without accepting input, uploading files, generating output, scoring, classifying, storing data, calling APIs, or creating a public route.

## E. Prototype Location

Approved location: `_internal_prototypes/evidence-posture-workbench/`

Files:
- `_internal_prototypes/evidence-posture-workbench/index.html`
- `_internal_prototypes/evidence-posture-workbench/prototype.css`

The location is repository-visible, non-public as a web surface, not linked from public pages, not included in sitemap.xml, not registered as a public route, and does not require .nojekyll.

## F. Prototype Visual Thesis

The prototype expresses Hoax.ai's validated conceptual interface direction: evidence chamber not detector; governed evidence field not black cyber dashboard; artifact first; boundary first; posture before verdict; uncertainty before confidence; provenance shadow before conclusion; missing context as absence; not assessable as protected restraint; refusal as governance; output envelope containment; no verdict gravity.

## G. Prototype Static Surface

Eight static, non-operational zones: Workbench Boundary Header, Artifact Context Zone, Source and Provenance Context Zone, Missing Information Zone, State Routing Zone, Refusal and Boundary Zone, Output Envelope Preview Zone, Verification Questions Zone.

## H. Prototype Component Families

Static visual representations only: Boundary Notice, Artifact Context Card, Source Context Card, Provenance Gap Indicator, Missing Information Register, State Status Badge, Refusal Reason Panel, Output Boundary Panel, Not Assessable Panel, Verification Question List.

## I. Prototype Content Rules

Fictional placeholder labels only. No real people, companies, institutions, brands, political events, current events, accusations, external factual claims, uploaded media, generated analysis, fake/real outputs, scores, percentages, verification claims, or detection claims.

## J. Prototype Implementation Rules

Allowed: static HTML, static CSS, semantic sections, accessible labels, inert visual panels, local CSS file.

Forbidden: JavaScript, forms, inputs, buttons implying action, upload controls, network calls, API calls, storage, analytics, generated outputs, dynamic scoring, interactive classification, external scripts, external libraries.

## K. Background Identity Rules

Governed evidence field direction: pale document field, bone neutral, warm ash, muted graphite, faint blue-grey provenance lines, restrained amber/copper boundary accents. Not generic black cyber dashboard, neon detector UI, red/green fake-real interface, or SaaS analytics gradient.

## L. Public Surface Boundary

The public website remains exactly the current public surface. Sitemap unchanged. Route registry unchanged. Public navigation unchanged. Prototype is not a launch, product, public workbench, detector, or engine.

## M. Current Status

Prototype v1 exists as static internal artifact. Workbench interface does not exist as public surface. Workbench engine, classifier, upload workflow, and scoring do not exist. Public workbench route does not exist. Public engine remains blocked.
