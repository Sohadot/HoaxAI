# Controlled Internal Prototype v0 — Boundaries

## Non-Public Rule

This internal prototype **cannot be linked publicly**, **cannot be deployed**, and **cannot be used with real claims**.

## Operational Prohibitions

- Cannot accept uploads
- Cannot generate reports or result cards
- Cannot generate public explanations or explanation UI
- Cannot publish benchmarks, coverage reports, or performance rankings
- Cannot expose API, UI, JavaScript, scoring, or analytics behavior
- Cannot call external data sources or live web services

## Governance Rule

This prototype **cannot be converted into a public engine** without future governed sprint authorization.

## Failure Condition

**Any public exposure fails the prototype boundary** and requires immediate rollback.
